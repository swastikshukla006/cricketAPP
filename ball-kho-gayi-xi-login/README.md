# Ball Kho Gayi XI — Player Login Edition

A mobile-first school cricket website with role-based local accounts.

## New account features
- Player login with a personal 4–6 digit PIN
- Personal dashboard with runs, wickets, catches, strike rate and economy
- Upcoming-match card and availability response
- Recent-form bars, best recent performance and team rankings
- Personal cricket target/goal
- Captain dashboard with wins, win rate, team runs, current form and squad availability
- Captain can schedule matches, upload results and update the captain's notice
- Administrator can manage players, login PINs, captaincy, backups and all website data
- Change-PIN flow for every logged-in account

## Important
This is a static HTML/CSS/JavaScript website. Login and data are stored in the browser, so this is suitable for testing and a trusted school group, but it is not secure server-side authentication. A shared production login requires a backend/database such as Supabase or Firebase.
