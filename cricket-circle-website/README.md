# Cricket Circle Website

A polished, mobile-first cricket group website built with plain HTML, CSS and JavaScript.

## Open it

Open `index.html`. It is now completely self-contained: the complete UI, styling and JavaScript are embedded inside this one file. No local server is required for the visual interface.

## Included

- Responsive dashboard with a full visual UI
- Match creation and fixture list
- Mobile live scoring with undo
- Player management and filtering
- Season leaderboard
- Expense and collection tracking
- Admin settings
- Local browser storage
- JSON data export

## Deploy to Netlify

Upload the folder or ZIP to Netlify. `index.html` is the entry file.

## Data note

This first version stores data in the browser using `localStorage`. Shared accounts and multi-device data will be connected to a backend in the next stage, before converting the website into an APK.
