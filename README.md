# ProProjects.co — Smooth Mobile Build v22

## Run locally

```powershell
npm install
npm run dev
```

Open `http://localhost:3000`.

## Included in this build

- Responsive mobile navigation
- iPhone-friendly Grainient rendering with reduced DPR and capped frame rate
- Reduced-motion accessibility support
- Batched GSAP reveals for smoother scrolling
- Mobile hero image grid instead of overlapping desktop cards
- Touch-friendly category filters
- Full-screen gallery image preview
- Product-specific enquiry form prefill
- Compressed review image uploads before localStorage saving
- Floating WhatsApp button on mobile
- Optimized image files without removing content
- Admin analytics, product management, enquiry handling, and review approval

## Secret admin access

- Open `/admin`
- Click the logo five times quickly
- Or press `Alt + Shift + O`
